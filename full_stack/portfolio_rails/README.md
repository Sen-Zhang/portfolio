### Rails Full-stack
#### Requirement
  - bower
  - ruby 2.3.1
  - bundler
  - PostgreSQL
  
#### Setup
1. Set up environment.

        % bin/setup

2. Verify that the app is up and running.

        % open http://foo.app.lvh.me:3000
